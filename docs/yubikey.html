Sécurité matérielle

Accueil Installation Sécurité

1. Authentification MFA (PAM)

Configuration du module Challenge-Response pour l'ouverture de session.

sudo apt install libpam-yubico yubikey-manager

2. Chiffrement LUKS

Liaison de la clé de déchiffrement du disque à la YubiKey.

sudo cryptsetup luksAddKey /dev/sdXn /etc/yubico/boot.response

3. SSH & GPG

Les clés privées ne quittent jamais la YubiKey.

gpg --card-status
