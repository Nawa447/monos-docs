Authentification & Chiffrement

Accueil Installation (Build) Sécurité (YubiKey)

1. MFA Locale (PAM)

sudo apt install libpam-yubico yubikey-manager
# Configuration du Challenge-Response
ykpersonalize -2 -ochal-resp -ohmac-sha1

2. Chiffrement LUKS

Lier le déverrouillage disque à la YubiKey :

sudo cryptsetup luksAddKey /dev/sdXn /etc/yubico/boot.response

3. GPG & SSH

Utilisation de la YubiKey comme Smartcard pour sécuriser vos connexions distantes sans stocker de clés privées sur le disque.
