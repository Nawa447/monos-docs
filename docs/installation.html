Guide de Création MonOS

Accueil Installation Sécurité

1. Préparation de l'environnement

Le build s'effectue dans une VM Debian avec l'outil live-build.

sudo apt update && sudo apt install live-build

2. Configuration du projet

Initialisation de l'arborescence :

lb config --distribution bookworm --binary-images iso-hybrid

3. Génération de l'ISO

Une fois les paquets listés dans config/package-lists/, lancez la compilation :

sudo lb build
