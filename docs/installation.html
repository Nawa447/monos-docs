Compilation de l'ISO

Accueil Installation Sécurité

1. Installation des outils

sudo apt update && sudo apt install live-build
Nous installons live-build, l'outil officiel de Debian pour transformer un système Linux en image ISO bootable.

2. Initialisation

lb config --distribution bookworm --binary-images iso-hybrid
Cette commande prépare l'arborescence. On choisit Bookworm (Debian 12) et le format iso-hybrid qui permet de démarrer l'ISO aussi bien sur une clé USB que sur un DVD ou une VM.

3. Lancer la génération

sudo lb build
C'est ici que la magie opère. Le script télécharge tous les composants de Debian et les compresse dans un seul fichier final nommé live-image-amd64.hybrid.iso.
