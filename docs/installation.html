Guide de Compilation

Accueil Installation (Build) Sécurité (YubiKey)

1. Préparation du projet

sudo apt update && sudo apt install live-build
mkdir mon-os && cd mon-os
lb config --distribution bookworm --binary-images iso-hybrid

2. Personnalisation

Ajoutez vos paquets dans config/package-lists/base.list.chroot :

vim curl git sudo openssh-server network-manager

3. Génération de l'ISO

sudo lb build

Le fichier généré est : live-image-amd64.hybrid.iso
