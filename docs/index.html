Accueil Installation Sécurité

Bienvenue sur la documentation

Ce projet vous guide dans la création d'une version de Debian personnalisée où l'accès est protégé par une YubiKey.

Pourquoi ce projet ? L'objectif est d'allier la stabilité de Debian à la sécurité du chiffrement matériel, rendant vos données inaccessibles sans votre clé USB de sécurité.
